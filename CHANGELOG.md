# Changelog

## [1.0] - 2026-06-14

### zh-Hans
- 增强了扫描和识别阶段的检查点功能。
- 新增了冻结扫描窗口功能。
- 添加了照片扫描检查点功能。

### en-US
- Improved checkpointing for scan and recognition phases.  
- Introduced a frozen scan window feature.  
- Added photo scan checkpointing functionality.

## [1.0] - 2026-05-30

### zh-Hans
- 增强重新识别功能并改善OCR验证逻辑。
- 添加可缩放照片视图以提升全屏照片交互体验。
- 修复手势释放时的闪烁问题。
- 记录全屏照片视图的缩放/平移决策。

### en-US
- Enhanced re-recognition functionality and improved OCR validation logic.  
- Added ZoomablePhoto view for better image interaction in FullPhotoView.  
- Resolved flickering issue on gesture release.  
- Improved alert presentation and updated changelog.

## [1.0] - 2026-05-24

### zh-Hans
- 增加健康数据写入功能，支持体重同步和健康报告重解析。
- 实现单图导入功能，提供便捷的菜单操作。
- 更新应用名称为“身记”，并增强OCR修正功能。
- 添加隐私政策页面，并在设置中集成隐私视图。
- 增加多语言本地化支持和使用指南。
- 提供联系信息的支持页面，方便用户获取帮助。

### en-US
- Added HealthKit integration for weight synchronization and user control settings.
- Implemented single-photo import functionality with a menu-based interface.
- Enhanced OCR correction features for improved diagnostics and misread handling.
- Added a support page with contact information and usage guidelines.
- Renamed the app from "我的身体" to "身记" for a fresh identity.
- Introduced multi-language localization support for a wider audience.
- Implemented a PrivacyView in Settings with an updated privacy policy.
